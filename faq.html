Questions

Before you buy

Is this a full catering CRM?

No. BanquetScale does one job: turning your recipes into accurate, scaled prep sheets for the guest count you actually have. If you need booking, invoicing, or staff scheduling too, run it alongside whatever CRM you already use.

Is this a subscription?

No. You pay once and keep the file. There's no recurring charge and no license check that can lock you out.

Do I need to install anything?

No. BanquetScale is a single file that opens in any modern browser, on desktop or tablet.

Can I reuse a menu across different events?

Yes. Build a full banquet menu once, save it as a group, and reuse it for the next event, at that guest count or a completely different one.

What if I need it on more than one device?

The file works standalone on any device. If you want live syncing between devices (say, the office computer and a tablet on the line), connect your own free Firebase project in a few minutes; setup steps are built into the app.

Where does my data live?

In your browser by default. If you turn on sync, it lives in the Firebase project you control, never on our servers.

What's the difference between the two tiers?

The file and features are identical. Single Kitchen covers one property; Multi-Location covers a catering company running the tool across several.
